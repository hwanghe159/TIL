# :bulb: 블로그 팁들 (티스토리)

### 커스텀 글꼴 특정 태그에만 적용하기

1. 타이포라에 글 작성

2. 티스토리 HTML모드로 전환, 

3. ```CSS
   <article class="markdown-body">
     <style>
       @import url(//fonts.googleapis.com/earlyaccess/hanna.css);
       h1, h2, h3, h4, h5, h6 {
         font-family: hanna !important;
       }
     </style>
   
   /* 내용 */
   
   <article>
   ```

   위 코드 복붙 

4. 타이포라에서 Ctrl + A로 모두선택 후 편집 -> HTML 코드로 복사

5. 티스토리로 돌아가서 위 코드의 `/* 내용 */` 부분에 붙여넣기

6. 미리보기로 폰트 적용 여부 확인하고 게시
