# :computer: IntelliJ 팁 및 단축키(Windows)

## 팁

- 자동 코드 접기 해제하기

  File -> Settings -> Editor -> General -> Code Folding -> Imports나 One-line methods 등 기능 해제하고 싶은것들 체크 해제





## 단축키

- 컨트롤 쉬프트 F10 : 테스트
- 컨트롤 알트 L : 코드 정렬
- 디렉토리 선택하고 컨트롤 알트 L : 디렉토리 내부 파일 전부 정렬
- 알트 인설트 엔터 : 테스트 코드 생성
- 컨트롤 k : 커밋
- 컨트롤 쉬프트 k : 푸시
- 컨트롤 알트 c : 상수로 빼기
- 컨트롤 알트 m : 블럭 설정하고 메서드 추출
- 쉬프트 F6 : 이름 한번에 바꾸기
- 쉬프트 F10 : 직전 액션
- 컨트롤 D : 현재라인 바로 밑에 복사
- 컨트롤 Y : 현재라인 삭제
- 컨트롤 W : 커서에 있는 단어 선택
- 컨트롤 쉬프트 J : 라인 합치기
- 컨트롤 알트 T : 둘러싸기
- 컨트롤 쉬프트 T : 클래스 파일에서 바로 테스트 파일 만들기
- 알트 6 : TODO 



### 이동

- 컨트롤 ↑/↓ : 스크롤
- 컨트롤 ←/→ : 단어 단위 이동
- 컨트롤 B : 메서드 정의로 이동 (컨트롤 클릭도 가능)
- 컨트롤 컨트롤 ↑/↓ : 커서복사
- 알트 ↑/↓ : 메서드 단위 이동
- 알트 ←/→ : 상단 파일 목록 이동
- 컨트롤 알트 ↑/↓ : 없는듯
- 컨트롤 알트 ←/→ : 이전 커서 위치로 이동
- 컨트롤 쉬프트 ↑/↓ : 메서드 위치 이동
- Page Up : 파일의 시작부분으로 이동
- Page Down : 파일의 끝부분으로 이동
- Home : 라인의 시작부분으로 이동
- End : 라인의 끝부분으로 이동
- 알트 1 : 프로젝트 트리로 커서 이동 (ESC로 에디터로 복귀)
