# :computer: IntelliJ 단축키 모음 (Windows)



- 컨트롤 쉬프트 F10 : 테스트
- 컨트롤 알트 L : 코드 정렬
- 알트 인설트 엔터 : 테스트 코드 생성
- 컨트롤 B : 메서드 정의로 이동
- 컨트롤 k : 커밋
- 컨트롤 쉬프트 k : 푸시
- 컨트롤 알트 c : 상수로 빼기
- 컨트롤 알트 m : 블럭 설정하고 메서드 추출
- 컨트롤 알트 좌, 우 : 이전에 봤던 파일, 후에 봤던 파일 보기
- 쉬프트 F6 : 이름 한번에 바꾸기
- 쉬프트 F10 : 직전 액션
- 컨트롤 쉬프트 방향키 : 한 블럭 위치 이동
- 컨트롤 D : 현재라인 바로 밑에 복사
- 컨트롤 Y : 현재라인 삭제
- 컨트롤 W : 커서에 있는 단어 선택
- 컨트롤 쉬프트 J : 라인 합치기
- Fn 페이지 업 : 라인의 시작위치로 이동
- Fn 페이지 다운 : 라인의 끝위치로 이동
- 컨트롤 컨트롤 위/아래 : 커서 복사
- 컨트롤 알트 T : 둘러싸기
