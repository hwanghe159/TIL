# :bulb: 자바 소소한 팁들



#### 정렬(배열)

오름차순 : `Arrays.sort(arr);`

내림차순 : `Arrays.sort(arr, Comparator.reverseOrder());` (int[]이 아닌 Integer[]로 선언해야함)

</br>

#### 정렬(리스트)
오름차순 : `Collections.sort(list);`

내림차순 : `Collections.reverse(list);`

</br>

#### 정렬(객체의 리스트)

```java
public class Person {
    private String name;
    private int age;
}
```

일때, 나이 순으로 `List<Person> people = ...`을 오름차순 정렬하고 싶으면

```java
List<Person> orderedPeople = people.stream()
                .map(Person::getAge)
                .sorted(Comparator.comparingInt(Person::getAge))
                .collect(Collectors.toList());
```

</br>

#### 초기화(배열)

`Arrays.fill(arr,값);`

</br>

#### String -> char 배열

```
String str = "abc";
char[] charArray = str.toCharArray();
System.out.println(Arrays.toString(charArray)); //[a, b, c] 이렇게 출력됨
```

</br>

#### String -> int

`int result = Integer.parseInt("123");`

</br>

#### int -> String

`String result = Integer.toString(123);`

</br>

#### String -> Boolean

`Boolean result = Boolean.parseBoolean("true") // "true"->true`

</br>

#### 문자열 뒤집기

```
String str = "abc";
String result = new StringBuffer(str).reverse().toString();
```

</br>

#### List<String> -> String

```
List<String> strings = ??;
String.join(", ", strings); //이렇게 하면 ", "으로 이어짐
```
